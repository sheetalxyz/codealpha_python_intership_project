# CodeAlpha Python Internship Project

## About

This repository contains the Python projects completed during my CodeAlpha Python Internship. These projects helped me improve my Python programming and problem-solving skills through hands-on practice.

## Projects

### 1. Basic Chatbot
A simple command-line chatbot that responds to predefined user inputs such as greetings and basic questions.

### 2. Hangman Game
A command-line Hangman game where the player guesses letters to complete a randomly selected word.

### 3. Stock Portfolio Tracker
A Python program that calculates the total investment value of a stock portfolio based on stock prices and user input.
